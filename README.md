Bpep
====

*Bpep is an Ethernet protocol Wireshark plugin.*

**This plugin is under construction.**

## Requirements

* *cmake*
* *wireshark*

## Build

```
$ git clone https://github.com/venthom/bpep.git
$ cd bpep
$ mkdir build
$ cmake ..
$ make
```

## Install

* Install the plugin in `~/.config/wireshark/plugins/`

```
$ make install
```
